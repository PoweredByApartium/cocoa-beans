Introduction

A CollectionObservable is an observable that wraps around a collection (like a list or set) and tracks changes to its contents. It differs from ğŸ”¬ Compound Observable by allowing the length of the collection to change dynamically, at the expense of having a single element type.

Most observables only react when their value changes—for example, when you assign a new list. But what if you just add or remove an item from the existing list? That's where CollectionObservable shines:

  • It knows when items are added, removed, or reordered.

  • Even if the reference stays the same, it can still detect internal changes.
