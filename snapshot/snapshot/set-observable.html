🧩 Set Observable

Available Since 0.0.39

A SetObservable is a specialized CollectionObservable that provides java.util.Set specific operations.

@Test void setExample() { SetObservable<String> names = Observable.set(); assertTrue(names.add("ikfir")); assertFalse(names.add("ikfir")); assertEquals(Set.of("ikfir"), names.get()); assertTrue(names.get().contains("ikfir")); }
