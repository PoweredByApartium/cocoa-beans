Introduction

Mapped observable is a reactive transformation layer, apply a transformation function to values from its source. This is the core of reactive mapping: taking one observable and transforming it into another value that updates automatically.

Usage

"even" : "odd" ); assertEquals("even", isEvenParity.get()); number.set(16); assertEquals("even", isEvenParity.get()); // Output "even" and isEvenParity won't be recompute because isEven at come up with same value }
